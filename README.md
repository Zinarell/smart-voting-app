# Скачивание модулей для фронтенда
1. Переходим в папку SmartVotingApp в проекте: `cd SmartVotingApp`
2. В консоли пишем `npm install`
3. Готово

# Скачивание библиотек для бэкенда
1. Переходим в папку smart-voting-backend в проекте: `cd smart-voting-backend`
2. Создаём виртуальное пространство: `python -m venv .venv`
3. Активируем виртуальное пространство: `.\.venv\Scripts\activate`
4. В консоли пишем `pip install -r requirements.txt`
5. Готово
